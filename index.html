Definition List

HTML
HyperText Markup Language, bahasa standar untuk membuat halaman web.
CSS
Cascading Style Sheets, bahasa untuk mendesain dan menghias tampilan halaman web.
JavaScript
Bahasa pemrograman yang digunakan untuk membuat halaman web lebih interaktif.
